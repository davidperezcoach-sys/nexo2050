# 🔄 Cómo usar este repositorio

## El puente David AI ↔ Nexo 2050

### Paso 1 — David AI escribe aquí
Cada vez que hagamos un cambio en Nexo 2050, yo (David AI) subo el código a este repo.

### Paso 2 — Tú vas al archivo
1. Abre el repo: https://github.com/davidperezcoach-sys/nexo2050
2. Entra en `pages/` o `functions/`
3. Abre el archivo que necesitas
4. Pulsa **Raw** (arriba a la derecha)
5. Ctrl+A → Ctrl+C

### Paso 3 — Pegas en el editor
1. Abre Nexo 2050 en Base44
2. Ve a la página correspondiente
3. Ctrl+A → Delete → Ctrl+V
4. Publicar ✅

## 📁 Estructura
```
pages/         → Páginas JSX (una por página de la app)
functions/     → Backend TypeScript  
components/    → Componentes reutilizables
```
